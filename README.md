# Star-Classification-using-ML-Models-Gaia-DataMining-Project-
Machine learning project to classify stellar spectral types (A/B) using Gaia dataset, applying multiple models (Logistic Regression, Random Forest, SVM, XGBoost) with hyperparameter tuning and achieving ~99.7% accuracy.

The goal of this study was to create a classifier that could predict the "SpType-ELS" property,
which denotes the estimated spectral class determined by Gaia and is classified as either "A" or
"B." It used a large dataset of stellar attributes to test several classification methods, such as
Logistic Regression, K-Neighbors, Random Forest, Decision Tree, Support Vector Classifier, and
XGBoost Classifier. The best-performing classifier was found through extensive data preparation,
hyperparameter adjustment, and model assessment. With its great accuracy, the final model proved
to be a useful tool for data mining in astrophysical applications and offered insightful information
on star classification. This research highlights the significance of machine learning to the
advancement of astronomy.
